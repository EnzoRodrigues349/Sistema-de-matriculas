 Sistema de Matrículas:

Sistema web desenvolvido com HTML, CSS e JavaScript, criado com o objetivo de simular um sistema de matrículas online de forma prática e responsiva.

 Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Funcionalidades
- Interface responsiva
- Sistema de cadastro de alunos
- Organização de informações em tela
- Interação com o usuário utilizando JavaScript

 Projeto Online
https://enzorodrigues349.github.io/Sistema-de-matriculas/

 Objetivo do Projeto
Este projeto foi desenvolvido para praticar conceitos de desenvolvimento web, estruturação de páginas, estilização com CSS e manipulação de elementos utilizando JavaScript.
